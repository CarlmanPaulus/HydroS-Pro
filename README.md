# HydroS-Pro v1.2.2

Hydrology tools for TROPI sites:
- **Manual Extractor** — extract manual water table data into split CSV based on site/pipes
- **RAW Combiner** — combine HOBO RAW logger files per site
- **RAW Combiner** — combine raw diver, baro and manual water table into processing excel
